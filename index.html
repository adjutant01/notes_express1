Net Ninja Express Basic

Home
  1. Check express version then install ---> npm express -v / npm install express
  2. Require 'express' ---> const express = require('express')
  3. Create express app ---> const app = express()
  4. Listen for requests ---> app.listen(port number)
  5. Use a 'get' request on the express app --->
    app.get(url, (req,res)=>{

    res.send(content)
    // or
    res.sendFile(absolute file path)
    //or
    res.sendFile(relative file path, {root:__dirname})
    })
  6. Install EJS ---> npm install ejs,
  7. Set it to the app ---> app.set('view engine', 'ejs')
  8. Set up folder if default isn't 'views' folder ---> app.set('views', 'folder name')
  9. Replace '.html' extension with 'ejs' ---> index.ejs
  10. Replace 'sendFile' with 'render' within the 'get' function ---> res.render(name of file)
  11. Pass title to the pages ---> app.get(url,(req, res)=>{res.render(filename, {title: name})})
  12. Use title: <%= title %>
  13. Add blogs to the 'home page' ---> const blogs =[{title:...,snippet:...},{''},{''}]
  14. Render it ---> res.render(page, {title:name, blogs:blogs})
  15. Inside 'home page', check to see if there are blogs before rendering them ---> <% if(blogs.length>0){%> <%}%>
  16. Create 'partials' folder. Create partials file with 'ejs' extention
  17. Place in desired location ---> <%- include(relative path) %>
  18. Server static 'style.css' file ---> app.use(express.static(foldername))
  19. Grab link from MongoDB
  20. Install Mongoose ---> npm install mongoose
  21. Require it then put it in a const ---> const mongoose = require('moongoose')
  22. Connect to the database using the 'connect' method. ---> mongoose.connect('the big link from MongoDB', {useNewUrlParser: true, useUnifiedTopology: true}).then(a=>{console.log('connected')}).catch(a=>{console.log(a)})
  23. Insert 'app.listen' into the mongoose connect function, to avoid loading page before data has been supplied.
  24. Create schema and model for data, put it inside a folder 'models'
  25. Inside folder, create 'blog.js', then require Mongoose inside file ---> require('mongoose')
  26. Grab 'Schema' from mongoose object and put it in a variable. ---> Schema = mongoose.Schema; (constructor function)
  27. Create a new schema, put it in constant. --->
    blogSchema = new Schema({
    title: {
    type:String, required: true
    },
    snippet:{
    type: String,
    required: true }, body:{ type:String,
    required:true
    } }, {timestamps: true})
  28. Create Model, put inside const ---> Blog = mongoose.model('name',blogSchema);
  29. Export. ---> module.exports = Blog
  30. Import module from models folder ---> const Blog = require('./models/blog')
  31. In app file ---> app.get(url,(req,res)=>{
    const blog = new Blog({
    title: 'new entry',
    snippet:' pokemon rules'
    body:' pikachu power'
    }) })
