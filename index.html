Home
  1. Check express version then install ---> npm express -v / npm install express
  2. Require 'express' ---> require('express'). Store in a 'const'
  3. Create express app ---> 'express()'. Store in a 'const'.
  4. Listen for requests ---> 'const_name.listen(port number)'
  5. Use a 'get' request on the express app --->'const_name.get()'
  6. const_name.get(url, (req,res)=>{

    res.send(content)
    // or
    res.sendFile(absolute file path)
    //or
    res.sendFile(relative file path, {root:__dirname})
    })
  7. Repeat above process for each page.
  8. Pass a middleware through the 'use' function on the app ---> 'const_name.use((req,res)=>{ res.sendFile(path, {root:__dirname})})'
  9. Install EJS ---> 'npm install ejs',
  10. Set it to the app ---> 'const_name.set('view engine', 'ejs')
  11. Set up folder if default isn't 'views' folder ---> const_name.set('views', 'folder name')
  12. Replace '.html' extension with 'ejs' ---> index.ejs
  13. Replace 'sendFile' with 'render' within the 'get' function ---> res.render(name of file)
  14. Pass title to the pages ---> const_name.get(url,(req, res)=>{res.render(filename, {title: name})})
  15. Use title: <%= title %>
  16. Add blogs to the 'home page' ---> const blogs =[{title:...,snippet:...},{''},{''}]
  17. Render it ---> res.render(page, {title:name, blogs:blogs})
  18. Inside 'home page', check to see if there are blogs before rendering them ---> <% if(blogs.length>0){%> <%}%>
  19. Create 'partials' folder. Create partials file with 'ejs' extention
  20. Place in desired location ---> <%- include(relative path) %>
