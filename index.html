Net Ninja Express Basic

Home
  1. Install express: 'npm install express'
  2. Require 'express' using require('express'). Store in a 'const'
  3. Create express app using 'express()'. Store in a 'const'.
  4. Listen for requests using: 'const_name.listen(port number)'
  5. Use a 'get' request on the express app. 'const_name.get()'
  6. const_name.get(url, (req,res)=>{ res.send(content) // or res.sendFile(absolute file path) //or res.sendFile(relative file path, {root:__dirname}) })
  7. Create multiple pages, add a navbar and add links.
  8. Use 'get' function on each URL.
  9. Pass a middleware through the 'use' function on the app. 'const_name.use((req,res)=>{ res.sendFile(path, {root:__dirname})})'
  10. Install EJS: 'npm install ejs', then set it to the app: 'const_name.set('view engine', 'ejs');
  11. const_name.set('views', 'folder name')
  12. Create 'blogs - about - New Blog' ejs files instead of html files: 'index.ejs'
  13. Render 'the view' within the 'get' function: res.render(name of file)
  14. Pass title to the pages: const_name.get(url,(req, res)=>{res.render(filename, {title: name})})
  15. Use title: <%= title %>
  16. Add blogs to the 'home page': const blogs =[{title:...,snippet:...},{''},{''}]
  17. Render it: res.render(page, {title:name, blogs:blogs})
  18. Inside 'home page', check to see if there are blogs before rendering them: <% if(blogs.length>0){%> <%}%>
